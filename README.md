# SRM SWAP

**SRM SWAP** is a peer-to-peer trading platform built for **SRM University AP**. It’s a closed community where you can buy, sell, and trade items with verified students.

### Why SRM SWAP?
*   **For SRM Students Only**: You need an `@srmap.edu.in` email to join. No strangers, just us.
*   **Hybrid Trading**: Offer money AND items for a deal. Example: "₹500 + My old headphones".
*   **Live Chat**: Discuss details instantly in real-time chat rooms.
*   **Spotlight UI**: A cool, modern interface that feels like a "Cyber-Campus".

### Key Features
*   **Smart Search**: Find exactly what you need, like specific books or electronics.
*   **Freshers Mode**: A special filter for new students to find hostel essentials.
*   **Trust Scores**: Earn points for good trades so everyone knows you're legit.
*   **Safety First**: Admin verification keeps the platform secure.

### Tech Stack
*   **Frontend**: React, TypeScript, Tailwind CSS
*   **Backend**: Node.js, Express, PostgreSQL
*   **Real-time**: Socket.IO for instant messaging

### How to Run It
1.  **Clone the Repo**: Download the code to your computer.
2.  **Start the Backend**:
    *   Go to the `server` folder.
    *   Run `npm install`.
    *   Set up your `.env` file (Database URL, Secret Key).
    *   Run `npm start` (Runs on port 5000).
3.  **Start the Frontend**:
    *   Go to the `client` folder.
    *   Run `npm install`.
    *   Run `npm run dev` (Runs on port 5173).
